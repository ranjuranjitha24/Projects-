# 💊 Smart Tablet Reminder App

## 🚀 Features
- Health-condition based reminders
- Pre-alert (5 minutes before)
- Daily tracking
- React + FastAPI full stack

## 🛠 Tech Stack
- Frontend: React
- Backend: FastAPI
- Database: SQLite

## 📦 Setup

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
